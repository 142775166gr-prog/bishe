# 视频处理功能说明

## 当前实现

### 1. 本地文件上传
- 支持视频、音频、文档、图片四种类型
- 自动获取文件大小
- 使用FFmpeg获取真实视频时长

### 2. 远程URL分析
- 支持分析远程文件URL
- 自动获取文件大小和MIME类型
- 从URL提取文件名作为标题
- **使用FFprobe/FFmpeg获取真实媒体时长**
- 支持手动点击"分析URL"按钮或输入框失去焦点时自动分析

## FFmpeg安装要求

为了获取真实的视频时长，系统需要安装FFmpeg：

### Windows安装
1. 下载FFmpeg：https://ffmpeg.org/download.html
2. 解压到目录（如：C:\ffmpeg）
3. 将bin目录添加到系统PATH环境变量
4. 重启应用服务器

### Linux安装
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install ffmpeg

# CentOS/RHEL
sudo yum install ffmpeg
```

### Mac安装
```bash
brew install ffmpeg
```

## 使用方式

### 本地上传
1. 选择文件类型（视频/音频/文档/图片）
2. 点击"选择文件"上传
3. 系统自动填入文件信息和真实时长

### URL分析
1. 在URL输入框中输入远程文件地址
2. 点击"分析URL"按钮，或让输入框失去焦点
3. 系统使用FFmpeg获取真实时长和文件信息

## 技术实现

- **FFprobe优先**：首先尝试使用ffprobe快速获取时长
- **FFmpeg备用**：如果ffprobe失败，使用ffmpeg作为备用方案
- **错误处理**：如果都失败，返回null让用户手动输入

## 专业解决方案

如果需要准确获取视频信息，建议使用以下方案：

### 方案一：使用FFmpeg（推荐）

1. **安装FFmpeg**
   - Windows: 下载FFmpeg并配置环境变量
   - Linux: `sudo apt-get install ffmpeg`
   - Mac: `brew install ffmpeg`

2. **添加Java依赖**
   ```xml
   <dependency>
       <groupId>net.bramp.ffmpeg</groupId>
       <artifactId>ffmpeg</artifactId>
       <version>0.7.0</version>
   </dependency>
   ```

3. **使用示例代码**
   ```java
   private Integer getVideoDurationWithFFmpeg(File videoFile) {
       try {
           FFmpeg ffmpeg = new FFmpeg("/path/to/ffmpeg");
           FFprobe ffprobe = new FFprobe("/path/to/ffprobe");
           
           FFmpegProbeResult probeResult = ffprobe.probe(videoFile.getAbsolutePath());
           
           if (probeResult.getStreams().size() > 0) {
               return (int) probeResult.getStreams().get(0).duration;
           }
       } catch (Exception e) {
           e.printStackTrace();
       }
       return null;
   }
   ```

### 方案二：使用Xuggler

1. **添加依赖**
   ```xml
   <dependency>
       <groupId>org.apache.tika</groupId>
       <artifactId>tika-core</artifactId>
       <version>2.4.1</version>
   </dependency>
   <dependency>
       <groupId>org.apache.tika</groupId>

       <version>2.4.1</version>
   </dependency>
   ```

### 方案三：前端获取（HTML5）

也可以在前端获取视频信息：

```javascript
function getVideoInfo(file) {
    return new Promise((resolve) => {
        const video = document.createElement('video');
        video.preload = 'metadata';
        
        video.onloadedmetadata = function() {
            resolve({
                duration: Math.round(video.duration),
                width: video.videoWidth,
                height: video.videoHeight
            });
        };
        
        video.src = URL.createObjectURL(file);
    });
}
```

## API接口

### 本地文件上传
- `POST /upload/video` - 上传视频文件
- `POST /upload/document` - 上传文档文件  
- `POST /upload/image` - 上传图片文件

### 远程文件分析
- `POST /upload/analyze-url` - 分析远程文件URL
  - 请求参数：`{ "url": "文件URL", "contentType": 1 }`
  - 返回信息：文件大小、文件名、MIME类型、**真实时长**

## 注意事项

1. **FFmpeg依赖**：系统需要安装FFmpeg才能获取真实时长
2. **网络访问**：分析远程URL需要服务器能访问目标地址
3. **超时处理**：FFmpeg分析有超时机制，避免长时间等待
4. **格式支持**：支持FFmpeg支持的所有视频和音频格式
